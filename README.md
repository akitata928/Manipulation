# 🧠 現代操弄藍圖 (The Blueprint of Modern Manipulation)



## 📖 專案簡介
本專案將探討現代社會工程與心理操弄運作機制的**「五層級心靈控制架構」**，轉化為一個具備維基百科風格的單頁式互動網頁（Single-Page Application / Website）。

現代的影響力已從傳統的「說服」（透過邏輯與證據改變觀點）轉變為「重塑現實」（透過情緒與社會壓力製造認同）。本專案旨在透過清晰的排版與互動設計，解構這套「洗腦演算法」，幫助大眾建立「模式察覺（Pattern Awareness）」的能力，進而奪回獨立思考的自主權。

## ✨ 核心特色與功能
* **維基百科風格排版**：採用乾淨、高易讀性的版面設計，專注於知識的傳遞與吸收。
* **原生平滑滾動 (Smooth Scrolling)**：點擊目錄連結時，頁面會平滑過渡至指定章節，提升閱讀體驗。
* **動態目錄監聽 (ScrollSpy)**：使用輕量級 JavaScript 實作。當使用者向下捲動閱讀時，側邊目錄會自動高亮標示當前所在的章節。
* **響應式設計 (RWD)**：純 HTML/CSS 打造，不依賴外部龐大框架，在手機、平板與桌機上皆能完美呈現。
* **在地化語境**：繁體中文內容已針對台灣社會現況與網路生態（如：出征、帶風向、老屁股、抓戰犯等在地用語）進行語境優化，使理論更貼近生活體驗。

## 🏗️ 理論架構總覽
本網頁深入探討了操弄心智的五個階段：
1. **設計師 (The Architect)**：建構現實與議程設定。
2. **誘惑者 (The Seducer)**：劫持大腦杏仁核與情緒。
3. **混淆者 (The Confuser)**：製造資訊過載與邏輯故障。
4. **攻擊者 (The Aggressor)**：將部落主義武器化。
5. **獄卒 (The Jailer)**：利用沉默螺旋達成工程化服從。
