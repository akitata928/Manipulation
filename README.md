# 🕉️ 現代操弄藍圖：新興宗教與神壇經濟的五層社會工程
*(The Blueprint of Modern Manipulation: Cults & Spiritual Economy)*

## 📖 專案簡介
本專案將「現代操弄藍圖（五層級心靈控制架構）」應用於分析新興宗教、極端信仰團體（如台灣俗稱的「紫衣神教」）背後的社會工程學與商業直銷模型。

在這些高度個人崇拜的組織中，「信仰」被量化為可以交易的籌碼。操弄者將信徒的大腦視為可駭入的系統，透過精密的五層級藍圖，將人們在現實生活中的脆弱與焦慮，轉化為組織擴張的燃料。在這裡，奉獻財富與時間不再是個人修養，而是被制約為「靈性升等」的唯一途徑。本專案旨在解構這套「靈性直銷」的演算法。

## ✨ 核心特色與功能
* **左側固定導覽列 (Sidebar)**：採用類似 GitBook 的專業技術文件排版，提供清晰的閱讀脈絡。
* **原生平滑滾動 (Smooth Scrolling)**：點擊目錄連結時，頁面平滑過渡至指定章節。
* **動態目錄監聽 (ScrollSpy)**：向下捲動閱讀時，側邊目錄會自動高亮標示當前所在章節。
* **深度在地化語境**：結合台灣常見的新興宗教現象（如：感恩師父、業力引爆、紫衣制服、接引利他、弘法護持金等），精準剖析其背後的心理機制。
* **數理邏輯對照**：將宗教洗腦手法對應至數學模型（定義域限縮、效用函數扭曲、馬可夫鏈等），提供冷靜、客觀的分析視角。

## 🏗️ 宗教操弄五層架構總覽
1. **藍圖設計（建構神聖現實）**：設定絕對的靈性定義域，壟斷救贖的解釋權。
2. **誘捕機制（靈性定價）**：劫持情感，將「奉獻財富與時間」等價於「靈性提升」。
3. **認知故障（神秘主義隔離）**：用權威與神蹟壓制理性查證，神聖化高層的貪婪。
4. **側翼執行（上下線階層）**：導入直銷 KPI 制度，利用菁英階級升等激發內部競爭。
5. **心理囚禁（團體制約）**：透過制服、儀式與靈性恐懼（業力引爆），完成絕對的心理制約。
